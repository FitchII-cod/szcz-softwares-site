Politique de confidentialité

Collecte de données

Ce site web statique est conçu pour minimiser la collecte de données personnelles. Les informations que vous partagez via le formulaire de contact sont envoyées directement via votre client e-mail et ne sont pas stockées sur ce site.

Utilisation des données

  • Formulaire de contact : Les informations que vous fournissez (nom, e-mail, message) sont utilisées uniquement pour répondre à votre demande.
  • Cookies : Ce site utilise uniquement un cookie local (localStorage) pour mémoriser votre préférence de thème (clair/sombre). Aucun cookie de suivi ou analytique n'est utilisé sans votre consentement explicite.
  • Données analytiques : Aucune donnée analytique n'est collectée sans votre consentement.

Hébergement et sécurité

Ce site est hébergé de manière sécurisée. Les données ne sont pas partagées avec des tiers, sauf obligation légale.
