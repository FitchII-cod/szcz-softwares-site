A propos du projet

Le LLM Copilote Interne est un assistant IA deployable en interne qui exploite la technique RAG (Retrieval-Augmented Generation) pour repondre aux questions des utilisateurs en se basant exclusivement sur les documents prives de l'entreprise.

Le systeme indexe les documents (PDF, Markdown, bases de connaissances) via des embeddings vectoriels, puis utilise la recherche semantique pour extraire les passages pertinents avant de generer une reponse contextuelle. Un systeme d'audit permet de tracer chaque reponse jusqu'a ses sources, garantissant transparence et fiabilite.

Fonctionnalites cles

Pipeline RAG

Indexation automatique des documents, chunking intelligent, generation d'embeddings et stockage vectoriel pour recherche semantique.

Filtrage et permissions

Systeme de filtres par departement, type de document et niveau de confidentialite. Chaque utilisateur voit uniquement ce qu'il est autorise a consulter.

Audit des reponses

Chaque reponse generee est accompagnee de ses sources, avec liens vers les documents originaux et score de confiance.

API REST

API FastAPI documentee (Swagger), integrables dans les outils internes existants (Slack, Teams, dashboards).
