À propos du projet

NFT Access Pass est un système complet de contrôle d'accès basé sur la blockchain. Un portail web permet aux utilisateurs de mint des NFT qui servent ensuite de "pass" pour débloquer des fonctionnalités premium dans une application existante.

Le smart contract gère l'émission, la vérification de propriété et les différents niveaux d'accès. L'intégration côté application vérifie en temps réel la possession du NFT dans le wallet connecté, permettant un modèle de monétisation décentralisé sans intermédiaire.

Fonctionnalités clés

Portail de mint

Interface web pour connecter son wallet, visualiser les NFT disponibles et procéder au mint avec paiement crypto.

Smart contract

Contrat déployé sur Solidity/Solana gérant l'émission, les metadata, les royalties et les niveaux d'accès.

Feature gating

Vérification de la possession du NFT côté application pour débloquer des fonctionnalités premium en temps réel.

Dashboard admin

Interface d'administration pour suivre les mints, les holders, les revenus et gérer les niveaux d'accès.

Stack technique

  • Solana / Anchor
  • Ethers.js
  • Web3.js
  • React
  • IPFS
  • MetaMask / Phantom
  • Hardhat
