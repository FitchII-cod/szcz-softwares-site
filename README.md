# SZCZ Softwares — Site vitrine (statique)
Focus: web, apps mobiles, IA (LLM/RAG), blockchain, jeux indés —

## Personnalisation
- Modifier `config.json` (textes, services, projets, stack, KPI, canaux).
- Palette aux couleurs du logo dans `styles.css` (dark/light).
- Formulaire en `mailto:` → pour envoi serveur: Netlify Forms ou Formspree.

## Déploiement
- Copie/rsync sur VPS derrière Nginx/Traefik/Caddy, ou Netlify/Vercel/Pages.

## Structure
/assets • index.html • privacy.html • styles.css • script.js • config.json
